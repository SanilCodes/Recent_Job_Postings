# Job Market Dashboard — Nashik, Mumbai & Pune

Scraped 3,500+ job listings from LinkedIn and Naukri, cleaned the data in Python, and built a Power BI dashboard to analyze the data job market across Maharashtra.

**Pipeline:** Scrape → Clean → Visualize

## Stack
`Python` `Pandas` `Power BI`

## What the dashboard covers
- Avg salary by role (Data Analyst, Data Scientist, ML Engineer etc.)
- Top skills in demand
- Best companies hiring right now
- Experience requirements breakdown
- How fresh the listings are

Data was normalized — salaries converted to LPA, posted dates turned into relative labels, skills split and counted individually.
